Nanakorobi Yaoki - Branding & E-Commerce

Nanakorobi Yaoki (Bento-fou)

Branding • E-Commerce • UI/UX Design

  • Auteur : David D'AMORE
  • Année : 2018-2024

Nanakorobi Yaoki (Bento-fou) incarne une fusion entre gastronomie japonaise et stratégie de marque. En tant que designer sur ce projet entrepreneurial, j'ai façonné l'identité visuelle globale : du logotype au branding physique (merchandising, cartes de visite, flyers...).

Le volet e-commerce a fait l'objet d'une attention particulière sur le parcours utilisateur (UX). Initialement sous WordPress/WooCommerce, la migration vers Symfony/Sylius a permis d'optimiser le flux d'achat et la gestion des stocks.

Afin de compléter notre offre avec du merchandising, j'ai réalisé des créations cohérantes avec notre marque (via Illustrator et Photoshop). Nous avons pris le pari gagnant d'automatiser cette partie en intégrant une solution de Print-on-Demand (POD) à notre plateforme. Les commandes étaient directement imprimées et expédiées, garantissant ainsi une livraison rapide et personnalisée.

Zoo Arcadia - SPA avec dashboard

Zoo Arcadia

SPA • Admin UI • UX Design

  • Auteur : David D'AMORE

Création d'une application Web "Single Page Application" (SPA) développée from scratch pour gérer le site et l'administration du Zoo Arcadia. Le projet comprend deux interfaces distinctes : une interface publique immersive pour les visiteurs et un tableau de bord d'administration pour la gestion interne.

Cette application démontre la mise en place d'une architecture front-end avancée et l'intégration de bonnes pratiques de déploiement (DevOps) :

  • Routing JS personnalisé : Système de navigation dynamique côté client pour des transitions instantanées sans rechargement de page.
  • Gestion des accès : Simulation d'un flux d'authentification avec le LocalStorage offrant des interfaces dédiées (Admin, Employé, Vétérinaire... ID et mot de passe respectif: admin/0001, employe/0002 et vet/0003).
  • Conteneurisation (DevOps) : Projet entièrement packagé via Docker et servi de manière optimale par un serveur Nginx.
  • Conception UI/UX : Approche Mobile-First 100% responsive, réalisée sans framework (Vanilla CSS/JS).

Figma (Wireframes / Mockups / Prototypes) Admin UI UX Design Trello (Gestion de projet / Kanban) JavaScript Architecture SPA Routing client Responsive (Mobile-First) LocalStorage API Docker MongoDB

Ibaraki Discovery

React UI • Experience Design • UX Writing

Ibaraki Discovery est une application Web développée en React pour promouvoir le tourisme durable dans la préfecture d'Ibaraki (Japon) et désengorger les destinations surpeuplées.
